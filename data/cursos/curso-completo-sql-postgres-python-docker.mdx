Este é um curso completo e prático de SQL, projetado para levar você do nível básico ao avançado. Utilizando tecnologias modernas como PostgreSQL, Python e Docker, o curso foca na aplicação real dos conceitos de banco de dados. Você aprenderá a configurar um ambiente de desenvolvimento robusto, manipular dados, realizar consultas complexas e até mesmo criar automações dentro do banco de dados.

## O que você vai aprender
- **Infraestrutura:** Configuração de ambiente com Docker e Docker Compose para rodar PostgreSQL e pgAdmin.
- **Fundamentos de SQL:** Comandos DDL (Create), DML (Insert, Update, Delete) e DQL (Select).
- **Manipulação de Dados:** Inserção de dados em massa utilizando scripts Python e Pandas.
- **Consultas Avançadas:** Uso de Joins (Inner, Left, Full), Agregações (Count, Sum, Avg), Window Functions e CTEs (Common Table Expressions).
- **Programação no Banco:** Criação de Stored Procedures, Functions e Triggers para automação e validação de dados.
- **Análise de Dados:** Resolução de problemas de negócio reais através de consultas SQL otimizadas.

## Pré-requisitos
- Não é necessário conhecimento prévio em SQL.
- Noções básicas de lógica de programação são úteis.
- Computador com capacidade para rodar Docker (Windows, Linux ou Mac).

## Para quem é este curso
Iniciantes que desejam entrar no mundo dos dados, desenvolvedores que querem aprofundar seus conhecimentos em banco de dados, e profissionais de negócio que precisam realizar análises de dados independentes.
