Neste tutorial prático, você aprenderá passo a passo como desenvolver uma aplicação desktop real utilizando tecnologias web. O vídeo cobre desde a configuração inicial do ambiente e conceitos fundamentais do Electron até a finalização do projeto com a geração de um arquivo executável (.exe) para distribuição.

## O que você vai aprender
- Fundamentos e funcionamento do Electron
- Configuração de ambiente com Node.js e NPM
- Estrutura de processos do Electron (Main e Renderer)
- Criação e manipulação de janelas (BrowserWindow)
- Estilização de interfaces desktop com CSS (frameless window, drag regions)
- Integração de bibliotecas externas (ts-particles para animações)
- Empacotamento e geração do executável com Electron Packager

## Pré-requisitos
- Conhecimento básico/intermediário de HTML, CSS e JavaScript
- Node.js instalado na máquina
- Visual Studio Code ou editor de texto similar

## Para quem é este curso
Desenvolvedores Web que desejam expandir suas habilidades para o desenvolvimento Desktop reutilizando seu conhecimento em JavaScript, ou qualquer programador interessado em criar softwares instaláveis modernos.
