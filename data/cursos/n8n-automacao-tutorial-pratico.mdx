Neste tutorial prático do Código Fonte TV, você será introduzido ao **n8n**, uma ferramenta open source de automação de fluxo de trabalho (workflow) que permite conectar aplicativos, serviços e bancos de dados. O vídeo guia você desde a instalação local e em um servidor VPS até a criação de um bot funcional que monitora o status de sites e envia alertas caso algum esteja fora do ar.

## O que você vai aprender
- **Introdução ao n8n:** Entenda o que é e como funciona essa ferramenta de automação low-code.
- **Instalação e Configuração:** Como rodar o n8n localmente usando Docker Compose e como fazer o deploy em um VPS (usando Hostinger e Docploy).
- **Criação de Workflows:** Construção passo a passo de um fluxo de automação.
- **Manipulação de Arquivos:** Leitura e processamento de dados a partir de arquivos de texto/listas.
- **Requisições HTTP:** Como configurar nós para testar o status de URLs (HTTP Requests).
- **Lógica Condicional:** Uso de nós "If" para tratamento de erros e tomadas de decisão no fluxo.
- **Integração com APIs:** Configuração de credenciais e envio de mensagens automáticas via API do Telegram.

## Pré-requisitos
- Noções básicas de lógica de programação.
- Familiaridade com o conceito de APIs (Requisições HTTP).
- Conhecimento básico de Docker é útil, mas não obrigatório (o processo é demonstrado).

## Para quem é este curso
Desenvolvedores, profissionais de DevOps e entusiastas de tecnologia que desejam otimizar tarefas repetitivas, criar integrações entre serviços sem escrever muito código e aprender a hospedar suas próprias ferramentas de automação.
