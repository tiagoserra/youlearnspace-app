Esta é a primeira aula de um curso completo de React. O instrutor Matheus Battisti apresenta o projeto que será desenvolvido ao longo das aulas: o "Costs", um sistema de gerenciamento de projetos [00:00:54]. Além de mostrar o projeto final, a aula serve como uma introdução teórica ao React, explicando o que é a biblioteca e seus conceitos essenciais.

## O que você vai aprender
- Visão geral do projeto "Costs" que será construído no curso [00:00:54].
- O que é React: uma biblioteca JavaScript para criação de interfaces [00:06:14].
- Conceitos fundamentais:
    - SPAs (Single Page Applications) [00:07:03]
    - Arquitetura baseada em componentes [00:07:43]
    - JSX (HTML dentro do JavaScript) [00:08:22]
    - Virtual DOM [00:09:09]

## Pré-requisitos
- Node.js e NPM instalados [00:04:51]
- Um editor de código (VS Code é recomendado) [00:05:30]
- Conhecimentos básicos de HTML, CSS e JavaScript.

## Para quem é este curso
Desenvolvedores que desejam começar a aprender React do zero e entender como construir uma aplicação completa e prática.
