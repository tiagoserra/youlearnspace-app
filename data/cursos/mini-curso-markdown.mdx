Este minicurso do Código Fonte TV ensina a linguagem Markdown do zero [00:00:34]. Ele aborda desde o básico da formatação até usos mais avançados, mostrando por que o Markdown se tornou um padrão não apenas para arquivos README no GitHub [00:00:06], mas também em ferramentas de IA para a escrita de prompts [00:00:20]. O curso utiliza o VS Code com extensões para demonstrar todo o potencial da linguagem [00:04:52].

- O que é Markdown, sua história e onde é utilizado [00:01:04].
- Ferramentas para escrever Markdown (StackEdit, Google Docs, VS Code) [00:01:52].
- **Formatação de Texto:** Títulos (H1-H6) [00:07:51], negrito, itálico e riscado [00:12:38].
- **Estrutura:** Parágrafos, quebras de linha [00:16:03] e separadores (linhas horizontais) [00:38:42].
- **Listas:** Ordenadas (numeradas), não ordenadas e listas de tarefas (checkboxes) [00:16:45], [00:18:24].
- **Links:** Criação de links, links de referência e âncoras internas [00:20:20].
- **Mídia:** Inserir imagens [00:24:23] e incorporar vídeos (usando HTML) [00:28:56].
- **Código:** Blocos de código (com syntax highlight para várias linguagens) e código inline [00:13:59], [00:31:04].
- **Avançado:**
    - Citações (Blockquotes) [00:30:34].
    - Criação de Tabelas com alinhamento [00:33:29].
    - Inserir HTML para formatações complexas (ex: redimensionar imagens) [00:28:02], [00:43:44].
    - "Flavors" do Markdown (GitHub Flavored Markdown) e Emojis [00:46:30], [00:48:05].
    - **Bônus:** Criar fluxogramas e diagramas diretamente no Markdown usando Mermaid [00:49:49].
