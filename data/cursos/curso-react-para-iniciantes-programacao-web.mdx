Este curso introdutório foi desenhado para quem deseja dar os primeiros passos no ecossistema React de forma rápida. Em menos de uma hora, você passará da teoria à prática, entendendo como o React funciona, como configurar um ambiente de desenvolvimento moderno e como estruturar uma aplicação básica utilizando as melhores práticas atuais (como Componentes Funcionais e Hooks).

## O que você vai aprender
- **Configuração de Ambiente:** Como criar um projeto React performático utilizando o Vite.
- **Componentização:** Criação e reutilização de componentes funcionais.
- **JSX:** A sintaxe que une JavaScript e HTML, incluindo interpolação de variáveis.
- **Props:** Como passar dados e atributos entre componentes de forma dinâmica.
- **Estados (State):** Gerenciamento de memória interna dos componentes com o hook `useState`.
- **Eventos:** Manipulação de eventos de clique e formulários.
- **Roteamento:** Navegação entre páginas (SPA) utilizando a biblioteca React Router Dom.

## Pré-requisitos
- Editor de código instalado (VS Code recomendado).
- Node.js instalado na máquina.
- Conhecimento básico de HTML, CSS e JavaScript.

## Para quem é este curso
Desenvolvedores front-end iniciantes que já dominam o básico da web (HTML/JS) e querem aprender sua primeira biblioteca de interface, ou profissionais que buscam uma atualização rápida sobre os conceitos fundamentais do React moderno.
